# 🎯 Project Overview

Every business spends thousands (or millions) on marketing—but **which channels actually generate revenue?**

This project applies **Marketing Mix Modeling (MMM)** to analyze historical marketing spend across different advertising channels and estimate **how much each channel contributes to sales**.

Instead of relying on intuition, this project enables **data-driven budget allocation**, helping businesses maximize ROI and reduce inefficient marketing spend.

---

# 💼 Business Problem

Marketing teams often struggle to answer questions like:

- 📺 Is TV advertising worth the investment?
- 📻 Does Radio generate better ROI?
- 🖥️ Are Banner Ads undervalued?
- 💰 Where should next quarter's marketing budget go?
- 📈 How can marketing spend be optimized to maximize revenue?

This project answers those questions using **Business Analytics + Machine Learning**.

# 📉 Diminishing Returns Analysis

As TV spending increases, total sales continue to grow—but **each additional dollar spent generates progressively smaller returns**.

This indicates **marketing saturation**, where increasing investment produces less incremental value.

# ⚠ Model Limitations

Although the model performs well, it has several practical limitations:

- 📌 Correlation does not imply causation.
- 📌 Seasonality is not modeled.
- 📌 Competitor marketing activity is ignored.
- 📌 Adstock (carry-over advertising effects) is not included.
- 📌 Linear Regression assumes constant returns, whereas real-world marketing often exhibits non-linear saturation.
